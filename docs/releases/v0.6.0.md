# Odin's Cat 0.6.0

Релиз от 9 апреля 2026 года.

## Что нового

- Добавлен Android-first MVP режима раздельного туннелирования в отдельной вкладке `Split App`.
- Пользователь может выбрать установленные приложения, которые должны идти в обход VPN, а остальной трафик продолжает идти через VPN как обычно.
- Вкладка `Split App` встроена в нижний блок текущего UI без перегрузки основного flow включения VPN.
- Для списка приложений добавлены:
  - поиск
  - быстрый выбор приложений для bypass
  - сохранение состояния между сессиями
- Android runtime теперь применяет `exclude apps from VPN` для основных режимов:
  - `VLESS + REALITY`
  - `YANDEX EDGE`
  - `VK RELAY`

## Исправления

- Исправлен сбор списка установленных Android-приложений:
  - теперь клиент видит не только launcher-приложения
  - в списке корректно появляются `OZON`, `Госуслуги` и другие обычные установленные приложения
- Исправлен `Split App` sheet для длинных списков:
  - кнопка `Закрыть` больше не уезжает вместе со скроллом
  - нижний dock выровнен по отступам и ширине
- Android label приложения обновлён под релизную версию: `Odin's Cat 0.6.0`

## Что важно для MVP

В `0.6.0` split tunneling реализован именно как `exclude apps from VPN`.

Это значит:

- выбранные пользователем приложения идут в обход VPN
- весь остальной трафик остаётся внутри VPN
- reverse mode (`only selected apps through VPN`) в этот релиз не включён, чтобы не усложнять Android MVP и не ломать stable path
