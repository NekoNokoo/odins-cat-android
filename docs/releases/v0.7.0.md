# Odin's Cat 0.7.0

`0.7.0` закрепляет новый Android-first путь вокруг `Yandex edge -> xhttp -> origin` и доводит его до состояния, которое уже можно передавать в реальный полевой тест.

## Что главное в этом релизе

- `Yandex camo` стал основным whitelist-facing режимом.
- рабочий путь теперь строится как `device -> Yandex VM :443 -> xhttp -> origin -> internet`.
- direct bypass для российских сайтов добавлен не только для `Yandex edge`, но и для `VLESS REALITY` и `VK proxy`.
- обновлён стартовый экран приложения.
- улучшены Android runtime mapping, diagnostics и owner/import flow.

## Что это даёт на практике

- выше шанс пройти среды с жёсткими whitelist-ограничениями;
- российские сайты и сервисы могут открываться напрямую, без лишней поломки внутри VPN;
- тесты `VLESS REALITY`, `Yandex edge` и `VK proxy` теперь сравниваются честнее, потому что bypass-логика выровнена между режимами.
